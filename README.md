# BCIT jenkins-agent Extension 'Reactor'

## `docker pull bcit.io/jenkins-reactor`

## This is an extension of [jenkins/agent:latest-alpine-jdk21](https://hub.docker.com/r/jenkins/agent)

## Utilities added for our use cases

- fancy motd
- vault
- ruby
- python3
- jq
- yq
- pup
- minio cli
- openshift cli
- openldap-clients
